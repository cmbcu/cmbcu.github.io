3000788: Introduction to Computational Molecular Biology

Offered in 1st semester (Aug-Dec)

Taught in Thai and English (2022/2025 = EN, 2023/2024 = TH)

Materials are available on GitHub

Description
This course reviews the basics of probability, statistics, and omics technology, and then builds upon this foundation to visit key bioinformatic algorithms and pipelines in genomics, transcriptomics, and proteomics.

Hands-on trainings on sequencing data processing, such as differential expression and functional enrichment analyses, are provided through in-class workshops and problem sets.

The latter half of the course introduces the concept of machine learning, with particular focuses on unsupervised learning: dimensionality reduction and clustering. Python language will be used but no student is expected to be able to code from scratch.

Special topics include single-cell transcriptomics, biological networks, chromatin conformation analysis, and more.

Target Audience
Students curious about the mathematics and motivations behind key bioinformatic algorithms.

Students wanting to learn about omics technology and try analyzing omics data for the first times.

3011979: Practical Data Science and Machine Learning for Medical Imaging

Offered in 2021-2022. No longer active. Will be replaced by a new course soon.

Taught in Thai

Materials are available on GitHub

Description
This course explores the principles of machine learning and deep learning with focuses on applications in medical imaging and radiology

Extensive hands-on trainings on Python programming and machine learning model development, are provided through in-class workshops and problem sets based on real-world data.

Target Audience
Students wanting to be able to handle data analysis and build machine learning models with Python.

Although background in mathematics is not required, students who are comfortable with calculus and computational thinking will be able to appreciate the intuitions behind various techniques.

3050571: Practical Clinical Data Science

Offered as a 6-week elective course for 5th year medical student (Feb-Mar)

Taught in Thai

Materials are available on GitHub

Description
This course provide both theoretical and practical foundation for medical students to pursue data science career.

Intensive practical assignments on both Python practices, project-based internship with KCMH data team, and clinical dataset analyses.

Target Audience
Medical students who are interested in data handling and analysis and willing to invest time on Python practices.

MLRS: Machine Learning Research School

Available on YouTube for both 2019 and 2023 versions

Description
This summer school was offered in 2019 and 2023 to provide a broad overview of the current state-of-the-art of deep learning

The format is lecture-based with some demo sessions. Topics include core ML concepts, computer vision, natural language processing, reinforcement learning, causal inference, as well as other related topics.

Speakers include academia and industry experts from major tech company.

Target Audience
Students with some prior exposure to machine learning and deep learning who want to survey the keywords and research landscape.
